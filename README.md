# FTC Byte Decode

A repository for FIRST Tech Challenge (FTC) team 23739 Byte Brigade robotics team resources and code.

## Overview

This repository contains code and documentation for FTC robotics competitions.

## Getting Started

### Prerequisites

- Android Studio
- FTC SDK
- Java Development Kit (JDK)

### Installation

1. Clone the repository:
  ```bash
  git clone https://github.com/your-username/ftc-byte-decode.git
  ```
2. Open the project in Android Studio
3. Sync Gradle files
4. Connect your Robot Controller phone or Control Hub

## Project Structure

```
ftc-byte-decode/
├── TeamCode/          # Team-specific code and utilities
├── FtcRobotController/ # FTC SDK controller app

```
